# Citation Note

This package extends a pyCSH brick-based C-S-H generation strategy for
Zn-modified C-S-H static candidate construction, validation, screening, and
selected quasi-static deformation diagnostics.

The manuscript and any public archive should cite the appropriate sources for:

- the original pyCSH brick-based C-S-H generation strategy:
  `[pyCSH reference]`
- structural interpretation of Zn-modified C-S-H, including Q(1,Zn)- and
  Q(2p,Zn)-related environments:
  `[Morales-Melgares et al. reference]`
- CementFF/CementFF4 and CementFF4-Zn parameter sources:
  `[CementFF/CementFF4-Zn parameter reference]`

This note intentionally uses placeholders rather than inventing final reference
formats.
